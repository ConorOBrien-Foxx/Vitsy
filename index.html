About

Vitsy is a 1D, stack-based language. Originally written in Java by VTCAKAVSMoACE, Vitsy has now been ported to JavaScript, with the help of many people, including Conor O'Brien (github, codegolf), Maltysen (codegolf), LegionMammal978 (codegolf), and, of course, VTCAKAVSMoACE.
